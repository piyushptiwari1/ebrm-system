# ebrm-system

> **Energy-Based Reasoning Machine — the production pipeline.**

`ebrm-system` is the installable framework that wraps any base reasoner with six swappable layers: intent routing, hierarchical reasoning, adaptive Langevin compute, process reward, external verifiers, and self-consistency voting.

See [Architecture](architecture.md) for the design, [API reference](api.md) for module-by-module docs, and the ADRs for why it's structured this way.
